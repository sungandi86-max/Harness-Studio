# Harness Studio

Your AI Workspace.

Build once.
Reuse forever.

Harness Studio는 프롬프트를 저장하는 곳이 아닙니다.

AI와 함께 만드는 모든 프로젝트의 Workflow, Prompt, Knowledge, Assets를 하나의 작업 공간에서 관리하는 AI Workspace입니다.

---

## Why?

AI를 사용하다 보면

- 프롬프트는 여러 곳에 흩어지고
- 프로젝트마다 작업 방식이 달라지고
- 좋은 Workflow가 재사용되지 못하는 경우가 많았습니다.

My Harness Library는

> **좋은 AI보다 좋은 Workflow가 더 오래가는 결과물을 만든다.**

라는 생각에서 시작되었습니다.
