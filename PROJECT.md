# Project: Harness Studio

## Product Definition

Harness Studio is a personal AI workspace for preserving how a user works, not only what they asked an AI once.

## Slogan

Your AI Workspace

## Supporting Copy

Build once. Reuse forever.

## Core Objects

- Project
- Prompt
- Workflow
- AI Team
- Knowledge
- Asset
- Rule
- Next Step
- Activity

## MVP Goal

Create a local-first app that lets Sookang open a workspace, continue active projects, reuse prompts and workflows, record knowledge, manage URL-based assets, and back up or restore data as JSON.

## Non-Goals

- Login
- Supabase
- AI API integration
- File upload
- Multi-user sharing
- Sensitive student or health data storage

## Design Direction

Dark navy, focused, premium, Korean-first, friendly in small details. The work surface should feel like a calm personal studio rather than an admin CRUD dashboard.
